## 🤔 이게 뭔가요?

**Korea Stats MCP**는 AI 비서(Claude, ChatGPT 등)가 **대한민국 공식 통계**를 직접 조회할 수 있게 해주는 도구입니다.

### 이런 질문을 할 수 있어요:

| 질문 예시 | 얻을 수 있는 정보 |
|----------|-----------------|
| "한국 인구가 몇 명이야?" | 최신 인구 통계 |
| "서울 출산율 추이 보여줘" | 10년간 출산율 그래프 |
| "부산과 대구 실업률 비교해줘" | 지역별 비교 데이터 |
| "아파트 가격 통계 찾아줘" | 관련 통계표 목록 |

### 데이터 출처

모든 통계는 **[KOSIS 국가통계포털](https://kosis.kr)**의 공식 데이터입니다.
- 통계청, 한국은행, 각 부처의 공식 통계
- 약 7만개의 통계표
- 실시간 업데이트

## 💻 설치하기

### 준비물

1. **Cursor IDE** 또는 **Claude Desktop** (MCP 지원 AI 앱)
2. **Node.js 18** 이상
3. **KOSIS API 키** (무료 발급)

### Step 1: KOSIS API 키 발급받기

1. [KOSIS 공유서비스](https://kosis.kr/openapi) 접속
2. 회원가입 (무료)
3. "활용신청" 클릭 → 자동 승인
4. "신청현황"에서 **인증키** 복사

### Step 2: 프로젝트 설치

```bash
# 1. 저장소 클론
git clone https://github.com/your-repo/korea-stats-mcp.git
cd korea-stats-mcp

# 2. 의존성 설치
npm install

# 3. API 키 설정
# .env 파일 생성 후 아래 내용 입력
echo "KOSIS_API_KEY=여기에_발급받은_키_입력" > .env

# 4. 빌드
npm run build
```

### Step 3: Cursor에 연결하기

Cursor 설정 파일 (`~/.cursor/mcp.json`)에 추가:

```json
{
  "mcpServers": {
    "korea-stats": {
      "command": "node",
      "args": ["경로/korea-stats-mcp/dist/index.js"]
    }
  }
}
```

### Step 4: Cursor 재시작

Cursor를 재시작하면 AI가 한국 통계를 사용할 수 있습니다! 🎉

### 예제 1: 기본 통계 검색

```

1. 시도/인구동태건수 및 동태율(출생사망혼인이혼)
   - 기관: 국가데이터처

2. 출생아수 및 합계출산율
   - 기관: 통계청
   - 기간: 1970~2024

원하시는 통계표의 데이터를 조회해드릴까요?
```

## ❓ 자주 묻는 질문

### Q: 어떤 통계를 조회할 수 있나요?

KOSIS에 등록된 약 **7만개** 통계표를 모두 조회할 수 있습니다:
- 인구/가구
- 경제/산업
- 고용/노동
- 주거/부동산
- 교육/건강
- 환경/교통
- 사회/복지

### Q: 데이터는 얼마나 최신인가요?

KOSIS에 업데이트되는 즉시 조회 가능합니다. 
통계마다 갱신 주기가 다릅니다:
- 인구: 매월/매년
- 물가: 매월
- GDP: 분기별

### Q: 무료인가요?

네, 완전 무료입니다! KOSIS API도 무료로 제공됩니다.

### Q: 영어로 질문해도 되나요?

네, 영어 질문도 가능합니다. 단, 통계 데이터는 한국어로 제공됩니다.

## 🔧 문제 해결

### "API 키 오류"가 나와요

1. `.env` 파일에 API 키가 올바르게 입력되었는지 확인
2. API 키 앞뒤에 공백이나 따옴표가 없는지 확인
3. `npm run build` 다시 실행

### "통계를 찾을 수 없습니다"가 나와요

1. 다른 키워드로 검색해보세요
   - "출산율" → "출생", "합계출산율"
   - "실업률" → "실업", "고용"
2. 지역명을 추가해보세요
   - "서울 인구", "경기도 GDP"

### 데이터가 이상해요

1. 통계표마다 지역 코드 체계가 다를 수 있습니다
2. AI에게 "get_table_info로 코드 확인해줘"라고 요청하세요

### Cursor에서 MCP가 연결 안 돼요

1. Cursor 완전 종료 후 재시작
2. `mcp.json` 파일 경로 확인
3. Node.js 버전 확인 (18 이상 필요)
